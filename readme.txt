Investment Calculator for professionals & businesses with contribution frequency of Annually, Semi-Annually, Monthly, Weekly & Bi-Weekly. Supports 250+ languages, English and EU number formats, editable labels. Use this calculator plugin to quickly figure out how much money you will have saved up during a set investment period. First, enter your initial amount you have set aside, then enter the interest rate along with how long you intend to invest for.

== Description ==

Whether you're considering getting started with investing or you're already a seasoned investor, this investment calculator can help you or your web visitors figure out how to meet your goals. It can show how your initial investment, frequency of contributions and risk tolerance can all affect how your money grows.

== Installation ==

Installation of the plugin is really simple.

1. Install using one of these options:
    * Install directly from WordPress Admin panel: go to Plugins -> Add New -> Search for "Smart Investment Calculator", and click the Install button.
    * Download the ZIP manually from WordPress' plugins repository, and upload it through WordPress Admin panel: go to Plugins -> Add New -> Upload Plugin, browse to the downloaded Zip and upload it.
    * Download the ZIP, extract it and manually upload the extracted folder through FTP to the `/wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!
